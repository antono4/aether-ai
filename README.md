# ✨ Aether

Personal AI Super Intelligence — local memory, managed services where needed, simple and powerful.

## Quick Start
```bash
./scripts/bootstrap.sh
pnpm dev
```

## Structure
- \`apps/\` - Applications (desktop, cli)
- \`packages/\` - Shared libraries (core, voice, etc.)
- \`scripts/\` - Build and setup scripts

## Requirements
- Node.js >= 24.0.0
- pnpm >= 10.10.0
- Rust 1.93.0+
